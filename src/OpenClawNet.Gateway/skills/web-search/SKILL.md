# Web Search Skill

You have access to web search tools that allow you to search the internet for up-to-date information.

## Capabilities

- **Search the web**: Execute keyword or natural-language queries against a search engine
- **Fetch pages**: Retrieve the content of specific URLs
- **Summarize results**: Extract and summarize relevant information from search results

## Guidelines

- Use web search when the user asks about current events, recent data, or information that may not be in your training knowledge
- Prefer authoritative sources (official documentation, reputable news outlets, academic sources)
- Always cite the source URL when providing information retrieved from the web
- Do not search for or retrieve content that violates privacy or terms of service
- Summarize search results concisely; avoid dumping raw HTML or overly long excerpts
